# Student Result Management System

## Overview

The Student Result Management System is a web-based application developed using HTML, CSS, and JavaScript. It helps in managing student academic records, calculating results, assigning grades, and displaying performance reports through an interactive and user-friendly interface.

## Features

* Add new student records
* Update existing student information
* Delete student records
* Search students by ID or name
* Automatic total and percentage calculation
* Grade generation based on marks
* Result analysis and reporting
* Responsive and user-friendly interface
* Data storage using Local Storage

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage

## How It Works

1. Enter student details and marks.
2. The system automatically calculates:

   * Total Marks
   * Percentage
   * Grade
   * Result Status
3. Student records are stored locally in the browser.
4. Users can view, update, search, and delete records whenever required.
